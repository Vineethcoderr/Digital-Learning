# Vidya Setu - Digital Learning Platform
# Requirements and Dependencies

## System Requirements

### Minimum Requirements
- **Node.js**: v20 LTS or higher (v23.2.0 tested)
- **npm**: v9.x or higher
- **RAM**: 4GB minimum (8GB recommended for development)
- **Storage**: 500MB for dependencies + project files
- **Network**: Internet connection for initial setup and cloud sync

### For Android Development (Optional)
- **Android Studio**: Arctic Fox or higher
- **Android SDK**: API level 21+
- **Capacitor CLI**: v8.x

## Environment Variables Required

### Backend (.env in /backend)
```
# Database
MONGODB_URI=mongodb+srv://<username>:<password>@cluster.mongodb.net/Vidya Setu

# JWT Secret
JWT_SECRET=your_super_secret_jwt_key_here

# Email Configuration (for nodemailer)
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-specific-password

# Cloudinary (for media storage)
CLOUDINARY_CLOUD_NAME=your-cloud-name
CLOUDINARY_API_KEY=your-api-key
CLOUDINARY_API_SECRET=your-api-secret

# Server Configuration
PORT=5000
NODE_ENV=development

# CORS (Frontend URL)
FRONTEND_URL=http://localhost:5173
```

### Frontend (.env in /frontend)
```
# Backend API URL
VITE_API_URL=http://localhost:5000/api

# Socket.IO Server URL
VITE_SOCKET_URL=http://localhost:5000

# App Configuration
VITE_APP_NAME=Vidya Setu
```

## Installation Steps

### 1. Clone Repository
```bash
git clone <repository-url>
cd digital-learning-platform
```

### 2. Install Backend Dependencies
```bash
cd backend
npm install
```

### 3. Install Frontend Dependencies
```bash
cd frontend
npm install
```

### 4. Configure Environment Variables
Create `.env` files in both backend and frontend directories with the required variables.

### 5. Install FFmpeg (Required for Video Compression)
- **macOS**: `brew install ffmpeg`
- **Windows**: Download from https://ffmpeg.org/download.html
- **Linux**: `sudo apt-get install ffmpeg`

### 6. Install Capacitor (For Android Build - Optional)
```bash
cd frontend
npm install @capacitor/core @capacitor/cli @capacitor/android
npx cap init
npx cap add android
```

## Troubleshooting

### Common Issues

1. **Module not found errors**
   - Run `npm install` in the respective directory
   - Clear npm cache: `npm cache clean --force`
   - Delete node_modules and reinstall: `rm -rf node_modules && npm install`

2. **MongoDB connection errors**
   - Verify MongoDB URI in .env
   - Check network access in MongoDB Atlas
   - Ensure IP whitelist includes your current IP

3. **FFmpeg not found**
   - Install FFmpeg system-wide
   - Add FFmpeg to system PATH
   - Verify installation: `ffmpeg -version`

4. **Port already in use**
   - Change PORT in backend/.env
   - Update VITE_API_URL in frontend/.env accordingly

5. **CORS errors**
   - Verify FRONTEND_URL in backend/.env
   - Ensure frontend is running on the specified URL
